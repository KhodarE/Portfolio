Dictionary to AVL Tree


The Dictionary to AVL Tree program is a very sophisticated program as it reads text files, containing words to create an AVL Tree. It selects each word placing them into a leaf node, to check if the word is either going to be placed to the left, right or be selected as the node. It keeps traversing until all the words in the dictionary/text file are inserted into the AVL Tree.

The application also consists of the following functionalities:

- Load dictionary

- Output dictionary to a new file

- Print dictionary

- Print the AVL Tree

- Inpyut words to the dictionary
